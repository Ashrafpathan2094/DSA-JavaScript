Minimum Recolors
to get K Blacks

⏱ O(n) Time ◻ O(1) Space
💡
Core Idea

Slide a window of size k across the blocks string. Count how many W (white) blocks are inside the window — those are the ones you must recolor to B. Track the minimum over all windows.

01

Initialize — count Ws in the first window of size k.

02

Slide — add the new right element; remove the old left element that fell out.

03

Update min — after each valid window, record the minimum white count seen so far.

04

Return — min is the answer: fewest recolors needed.

Optimal Solution
var minimumRecolors = function(blocks, k) {
  let min = Infinity;       // ✓ cleaner than a magic number
  let whites = 0;

  for (let i = 0; i < blocks.length; i++) {
    // 1. Expand: add new right element
    if (blocks[i] === 'W') whites++;

    // 2. Shrink: remove element that fell off left
    if (i >= k) {
      if (blocks[i - k] === 'W') whites--;
    }

    // 3. Record min when window is full
    if (i >= k - 1) {
      min = Math.min(min, whites);
    }
  }
  return min;
};
🔑
Why Not Brute Force?

A brute-force approach checks every window from scratch — O(n·k). The sliding window reuses the previous count: just subtract what left and add what entered. One pass, constant memory — O(n).
