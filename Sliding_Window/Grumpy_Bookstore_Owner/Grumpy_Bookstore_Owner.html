📚 Leetcode #1052

Grumpy Bookstore Owner

Sliding Window · Prefix Optimization
Medium Sliding Window Array
Problem Statement

A bookstore owner has an array customers[] where each element is customers visiting at minute i. The owner is grumpy at times indicated by grumpy[] (1 = grumpy, 0 = not grumpy). When grumpy, customers are not satisfied. The owner can use a secret technique for minutes consecutive minutes to suppress grumpiness.

Return the maximum number of customers that can be satisfied.

💡
Core Insight
Split the problem into two parts: (1) customers already satisfied regardless of the technique (grumpy[i]=0), and (2) find the sliding window of length minutes that recovers the most additional customers from grumpy moments. These two parts are independent — compute them separately, then add.

Algorithm Walkthrough
1
Compute Base Satisfaction
Loop through all indices. Wherever grumpy[i] === 0, add customers[i] to baseSatisfied. These customers are always happy regardless of the technique.
2
Initialize First Window
Slide a window of size minutes starting at index 0. Sum up customers[i] where grumpy[i] === 1 — these are the extra customers we'd recover by using the technique here.
3
Slide the Window
For each new position, add the incoming element (if grumpy) and remove the outgoing element (if grumpy). Track maxGain = Math.max(maxGain, windowGain) at every step.
4
Return Result
The answer is baseSatisfied + maxGain. The best window position gives us the most bonus customers from the technique.
JavaScript 
var maxSatisfied = function(customers, grumpy, minutes) {
    // ① Always-satisfied customers (grumpy[i] === 0)
    let baseSatisfied = 0;
    for (let i = 0; i < customers.length; i++) {
        if (grumpy[i] === 0) baseSatisfied += customers[i];
    }

    // ② First window: how many grumpy customers we'd save
    let windowGain = 0;
    for (let i = 0; i < minutes; i++) {
        if (grumpy[i] === 1) windowGain += customers[i];
    }

    let maxGain = windowGain;

    // ③ Slide window: add new, remove old
    for (let i = minutes; i < customers.length; i++) {
        if (grumpy[i] === 1) windowGain += customers[i];
        if (grumpy[i - minutes] === 1) windowGain -= customers[i - minutes];
        maxGain = Math.max(maxGain, windowGain);
    }

    // ④ Base + best window bonus
    return baseSatisfied + maxGain;
};

Time
O(n)
Space
O(1)

Two linear passes: one to compute base satisfaction, one for the sliding window. No extra data structures needed — just a few counters. Optimal for this problem.
